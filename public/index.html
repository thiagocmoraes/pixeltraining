FB Pixel Practical Training

Welcome to Facebook Pixel Training! This material aims to help you understanding how to implement a pixel in a website and to get some information from the page dynamically.

The idea of this training is NOT to train you as a programmer, but in order to get you comfortable in discussing technical matters, the best way is to do a little coding by yourself.

How to approach the problems?

Most of the problems here have simple solutions, but there are ones that will require googling and looking into technical documentation. Learning how to find the right doc is a critical part of a programmer's job so try to avoid getting too many pointers on how to find the info. Try it yourself first!

There are six exercises related to implementing the pixel and one more related to Tag Management Systems. Try to do the first ones and take a look into the Tag Manager as well. Don't worry if you have no idea how to do some exercises, if you did all of them, you can probably apply for a pixel implementator posision in some of our clients. :)

Problem descriptions

1 - Implement FB Pixel from your ad account in this page. Don't forget to test using Pixel Helper

2-1 - Migrate from Conversion Pixel to FB Pixel. This page uses an old pixel. Create a new one on Ads Manager or add an existing one form your account.

2-2 - Migrate to FB Pixel and Put Price on Pixel. Like last one, this page uses an old pixel. Create a new one on Ads Manager or add an existing one from your account. Also, you should get the conversion value and send it in the pixel.

3 - DPA Time! Fire the ViewContent event passing the product ID. Check the docs to see the required parameters. Make sure the ID matches according to Pixel Helper.

4 - Order ID. Fire the Purchase event. Don't worry about product_ids, but pass the Order ID in the pixel. This one is a little harder. Code in previous exercises might help in this one :)

5 - Fire event on button click. Try to fire the pixel only after the user clicked on the button. Search the web to find how to do it. For FB-specific instructions, you may want to check Tag API.
